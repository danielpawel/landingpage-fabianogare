# ⚖️ Treinamento Elite: Cálculos e Perícias Trabalhistas — Fabiano Gare

![Landing Page Preview](https://via.placeholder.com/1200x630/050505/C9A84C?text=Fabiano+Gare+Landing+Page+2026)

Landing Page de alta conversão desenvolvida com foco em autoridade, precisão técnica e imersão visual para o curso de Cálculos Jurídicos do Perito **Fabiano Gare**.

## 🚀 Tecnologias & Design System (2026)

Este projeto utiliza o estado da arte em desenvolvimento frontend para garantir performance (Core Web Vitals) e conversão:

- **HTML5 Semântico:** Estrutura otimizada para SEO e acessibilidade.
- **Tailwind CSS v4:** Estilização moderna com design tokens personalizados.
- **Vanilla JavaScript:** Interatividade fluida, cronômetros persistentes e controle de VSL sem dependências externas.
- **Interatividade Cinética:** Efeitos de iluminação dinâmica (Glow Follow) e animações suaves de entrada.
- **VSL de Alta Retenção:** Autoplay inteligente com desmutagem na interação do usuário e barra de progresso customizada.

## 🛠️ Funcionalidades Premium

- **Urência Persistente:** Cronômetro de 24h que salva o progresso via `LocalStorage`, personalizando a escassez para cada visitante.
- **FAQ Interativo:** Accordion dinâmico para quebra de objeções de venda.
- **Prova Social Dinâmica:** Bento Grid de depoimentos com tratamento visual de elite.
- **Navegação Líquida:** Scroll suave e ancoragem inteligente para pontos de conversão.
